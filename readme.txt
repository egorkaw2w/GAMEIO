# GameIO — Защищённая система продажи игровых аккаунтов и ключей

## Описание проекта
Система позволяет продавать игровые аккаунты и ключи на платформах Steam, EA App, Epic Games и др.  
Поддерживает **RBAC**, **шифрование**, **аудит**, **аналитику**, **бэкапы** и **Docker**.

---

## Роли пользователей (RBAC)

| Роль | Права |
|------|-------|
| **admin** | Полный доступ: управление товарами, пользователями, отчётами |
| **manager** | Управление заказами, просмотр аналитики |
| **moderator** | Модерация контента, просмотр логов |
| **user** | Покупка, просмотр своих заказов |

---

## Архитектура (простыми словами)

> **Клиент** → **API** → **База данных** → всё в **Docker**

1. **Клиент (React + MUI)** — веб-интерфейс, работает в браузере.
2. **API (Node.js + Express)** — обрабатывает запросы, проверяет JWT, вызывает процедуры в БД.
3. **База данных (PostgreSQL)** — хранит всё: товары, заказы, пользователей, логи.
4. **Docker** — запускает всё одной командой: `docker-compose up`.

---

## Что где находится и зачем

| Папка | Содержимое | Зачем |
|------|-----------|-------|
| `db/` | SQL-скрипты | Создание таблиц, VIEW, процедуры, триггеры, бэкапы |
| `api/` | Express-сервер | Обработка запросов, JWT, вызовы к БД |
| `client/` | React-приложение | Интерфейс для пользователей |
| `.env` | Секреты | Пароли, ключи шифрования — **не коммитится!** |
| `docker-compose.yml` | Запуск | Поднимает 3 сервиса: БД, API, Frontend |

---
